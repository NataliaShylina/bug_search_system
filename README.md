# Bug Search and Analytics System

Semantic search and analytical system for bug reports (JIRA / Bugzilla).

## Project goal
The goal of this project is to build a search and analytics system that supports
bug triage and prioritization using NLP and information retrieval techniques.

## Key features
- Semantic search using SBERT embeddings
- TF-IDF lexical similarity
- FAISS vector indexing for fast top-k search
- Sentiment analysis of bug descriptions
- Bug classification (priority)
- PostgreSQL-based storage
- FastAPI backend

## Technologies
- Python
- FastAPI
- PostgreSQL
- Sentence-BERT (SBERT)
- FAISS
- scikit-learn

## Data
Public datasets:
- JIRA Public Dataset (Kaggle)
